FE - First Year

Semester 1

Engineering Mathematics 1 - Basics of calculus and algebra.

Engineering Physics - Fundamentals of physics for engineers.

Basic Electronics - Introduction to electronic components.

C Programming - Introduction to programming using C language.

Semester 2

Engineering Mathematics 2 - Advanced calculus and transforms.

Engineering Chemistry - Basic chemistry concepts.

Data Structures - Arrays, linked lists, stacks and queues.

Digital Electronics - Logic gates and digital circuits.

Discrete Mathematics - Set theory, graph theory and logic.

Semester 4

Engineering Mathematics 3 - Numerical methods and statistics.

Web Technology - HTML, CSS and JavaScript basics.
